# Коллайдер добра — прототип сайта фонда «Среда Возможностей»

Концепт-прототип премиального сайта благотворительного фонда нового поколения.

**Идея:** не «фонд жалости», а системная инфраструктура добра — прозрачность, масштаб и измеримый вклад. Тёмная статусная эстетика «коллайдера», без давления на эмоции.

## Что внутри
- Анимированный «коллайдер» на canvas и живой логотип-спираль
- Блоки: манифест, направления помощи, «Кнопка добра» (донат по ссылке), партнёрам, отчётность
- Интерактивный демо-калькулятор **«Твой процент вклада добра в мир»** + рейтинг социальной ответственности
- Адаптив, scroll-анимации, доступность (reduced-motion)

## Стек
Чистый статический сайт (HTML/CSS/JS, без сборки) — хостинг на GitHub Pages.

```
index.html
assets/css/style.css
assets/js/main.js
```

> Прототип. Оплата и расчёты — демонстрационные; на проде подключаются платёжная ссылка/эквайринг и ИИ-оценка вклада.
